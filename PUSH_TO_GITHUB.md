# Quick Guide: Push to GitHub

## Step 1: Add your GitHub repository as remote

Run this command (replace YOUR_USERNAME with your GitHub username):

```bash
git remote add origin https://github.com/YOUR_USERNAME/Zuno-Tools.git
```

## Step 2: Push your code

```bash
git branch -M main
git push -u origin main
```

That's it! Your code will be on GitHub.
